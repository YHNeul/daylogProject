# 📆SpringBoot - 캘린더(일정 관리) + Todo + 일기 통합 서비스

## ✔️소개

> 캘린더, Todo, 다이어리까지 한 번에 관리하는 올인원 생산성 웹 플랫폼

## 💻 기술 스택
### Backend
- Java 17
- Spring Boot 3.2.4
- Spring Boot Data JPA, Security
- Jwt
- Lombok
- MariaDB
- Maven
### Frontend
- React
- React Router, Bic Calendar
- Axios
- MaterialUI

## 💡 주요 기능

- 이메일 인증을 통한 회원가입
- 일일 일정, 기간 일정 관리
- 카테고리 별 색상 분류 (회사, 가족, 친구 등)
- Todo 기록 시 달성도 체크
- 다이어리 기록 시 이미지 업로드
